# 【架電メンバー向け】日報作成画面アップデートのお知らせ

架電日報の作成フローがアップデートされました。
主に **3つの変更点** がありますので、ご確認をお願いします。

---

## 1. シフト確認・修正フロー

架電日報の提出時に、**当日のシフト情報を確認するステップ** が追加されました。
日付とプロジェクトを選ぶと、登録済みのシフトが自動で表示されます。

| こんなとき | やること |
|---|---|
| シフト内容が合っている | 「**このシフトで報告する**」ボタンを押す |
| 時間が実際と違う | 「**修正する**」ボタン → 正しい時間を入力 → 「**この時間で確定**」 |
| シフトが未登録 | 開始・終了時間を入力 → 「**シフトを登録**」ボタンを押す |

確認が完了すると、緑色の **「確認済」** バッジが表示されます。

> シフトを確認しないと日報の提出ができません（下書き保存は可能です）。
> Googleカレンダーでシフトを変更した場合は、「**シフトを再取得**」ボタンで最新情報を取得できます。

---

## 2. 架電目標の自動計算と休憩時間の入力

架電数の目標は、確認したシフト時間をもとに **1時間あたり25件** で自動計算されます。

```
架電目標 ＝（シフト時間 − 休憩時間）× 25件/h
```

**休憩時間がある場合** は、シフト確認後に表示される「**休憩時間**」入力欄に分数を入力してください。
入力すると架電目標が自動で再計算されます。

> 例）シフト 9時間・休憩 60分 の場合：（9h − 60分）= 8h × 25件/h = **200件**

---

## 3. テレアポくんからの架電実績の自動取得

テレアポくんで架電している方は、日報のKPI欄に **架電実績をワンクリックで反映** できるようになりました。

### 手順
1. 日付・プロジェクトを選択
2. KPI実績セクション右上の **「テレアポくんから取得」** ボタンを押す
3. 架電数・通電数・アポ数が自動入力される

取得後に数値を手動で修正することも可能です。

---

## その他

- **下書き保存** ができます。途中で中断しても続きから入力できます。
- 画面下部に **当月のKPI累計**（総架電数・総通電率・総アポ率）が表示されます。

ご不明点があれば管理者までお気軽にお問い合わせください。
